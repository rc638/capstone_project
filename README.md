- [x] PlatformIO + ESP32-S3 개발 환경 설정

- [x] Node-RED 설치 및 대시보드 플러그인 설치
- [x] ESP32 → WiFi → MQTT → Node-RED 실시간 차트 연동 완료
- [x] 타격 감지 쿨다운 로직 추가 (중복 전송 방지)
